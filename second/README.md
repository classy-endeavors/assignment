# Task Management System - Design Patterns Demo

This project demonstrates various design patterns in JavaScript through a task management system.

## Design Patterns Used

1. **Singleton Pattern** - TaskManager ensures only one instance exists
2. **Factory Pattern** - TaskFactory creates different types of tasks
3. **Observer Pattern** - Notification system for task updates
4. **Strategy Pattern** - Different priority handling strategies
5. **Command Pattern** - Undo/redo functionality for task operations
6. **Decorator Pattern** - Task enhancements (tags, due dates)
7. **Facade Pattern** - Simplified API for complex subsystem interactions

## Running the Project

```bash
npm start
```

## Project Structure

- `index.js` - Main entry point
- `patterns/` - Design pattern implementations
- `models/` - Data models
- `utils/` - Utility functions
